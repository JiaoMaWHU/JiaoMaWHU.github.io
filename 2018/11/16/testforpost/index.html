Hexo Markdown 语法总结

Posted on 2018-11-16 | In Tutorial |

总结下Hexo Markdown的语法，发现很多东西都只支持原生markdown。

1.front-matter 格式:

2.斜体: *斜体* 粗体：**粗体**

3.分级标题：

使用下划线
使用#标识

4.分割线：在单独一行使用 -------

5.删除线: 使用~~删除线~~ 下划线:<u>balabala</u>

6.超链接：

7.注释: \注释文字

8.引用: >文字引用

9.脚注: BlaBlaBla [^anyword], [anyword^]: Bla means ...

10.无序列表 +：实心圆，-：空心圆，*：方形，有序列表：1. 2.

11.表格: 绘制表格格式如下，|控制分列，-控制分行，:控制对齐方式。

13.LaTex公式

数学公式的默认定界符是$...$和\\(...\\)（对于行内公式），以及$$ ... $$和\\[...\\]（对于块级公式）
