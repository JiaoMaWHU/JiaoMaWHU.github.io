Leetcode Solutions

Sum Topic

1.Two Sum

Given an array of integers, return indices of the two numbers such that they add up to a specific target. You may assume that each input would have exactly one solution, and you may not use the same element twice.

  • The best solution here is using a hash table. We cannot use a vector, because we do not know what the range of integers is. So we need unordered_map, where the key is integer and the value is index.

  • A trick: we can finish the initialization of map and the search in on loop, which is also solid when the two numbers are same.

15.3Sum

Given an array nums of n integers, are there elements a, b, c in nums such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero. Note: The solution set must not contain duplicate triplets.

  1. Why not use the same way as 2Sum? TLE
  2. Sort firstly, start from the first element i, use two pointers moving towards each other: i+1 -> <- size()-1.
  3. Be careful of the duplicate triplets.

167.Two Sum II - Input array is sorted

Given an array of integers that is already sorted in ascending order, find two numbers such that they add up to a specific target number. The function twoSum should return indices of the two numbers such that they add up to the target, where index1 must be less than index2.

Use two pointer from the start and the end of the vector to find the target pair. Time Complexity is O(n).

170.Two Sum III - Data structure design

Design and implement a TwoSum class. It should support the following operations: add and find. add - Add the number to an internal data structure. find - Find if there exists any pair of numbers which sum is equal to the value.

Use hashmap to store a element and the times it shows up.

560.Subarray Sum Equals K

Given an array of integers and an integer k, you need to find the total number of continuous subarrays whose sum equals to k.

  1. Use hashmap, the key is the sum at index at i, the value represents how many index has this sum.
  2. Therefore, we can solution this in O(n).

3. Longest Substring Without Repeating Characters

Description: Given a string, find the length of the longest substring without repeating characters.

  1. Apparently, we should use a hash table to store characters. Vector is fine, because we only have 128 characters in ASCII table. Then we can use a loop to find the substring.
  2. The trick is, when we meet a character c2 that repeats with c1 in the substring, we can jump into index(c1) +1 to reduce computation.

4. Median of Two Sorted Arrays

There are two sorted arrays nums1 and nums2 of size m and n respectively. Find the median of the two sorted arrays. The overall run time complexity should be O(log (m+n)). You may assume nums1 and nums2 cannot be both empty.

  1. Do binary search for K, which means to go over the K/2 numbers, then K/4, … in both of the vectors. Then, the complexity will be O(log(m+n)).
  2. use get_kth to represent the recursive function. We can use (v(size+1)/2 + v(size+2)/2)/2 as the result, which is both correct when the vector has even numbers or odd numbers.
  3. Then we start to search in both vectors, we find the No.K/2 number in both arrays at first, then change the index and find the No.K/4 number, …
  4. Be careful of lots of corner cases, such as the K/2 is already larger then the size of the vector, so we can directly find the result in the other vector.

5. Longest Palindromic Substring

Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.

  1. Use 2D dp matrix to store the maxLen, dp[i][i] = 1; dp[i][i+1] = 1, if string[i] == string[i+1]; dp[i][j] =1, if string[i] == string[j] and dp[i+1][j-1]. We use two loop, first i from 0->n, second j starts from 0 and ends at i.
  2. We can also simply starts from the center towards both sides, and be care that center can be a char, or between two characters.

54. Spiral Matrix

Given a matrix of m x n elements (m rows, n columns), return all elements of the matrix in spiral order.

  1. Iterative method is good, however it needs an extra matrix.
  2. Use recursive method, parameter i, j start pointer, n, m end row and column.

58. Length of Last Word

Given a string s consists of upper/lower-case alphabets and empty space characters ‘ ‘, return the length of last word in the string. If the last word does not exist, return 0.

Start from the end of the string and count the size will be.

71. Simplify Path

Given an absolute path for a file (Unix-style), simplify it. Or in other words, convert it to the canonical path.

The solution is simple, use stack to store the path.

89. Gray Code

The gray code is a binary numeral system where two successive values differ in only one bit. Given a non-negative integer n representing the total number of bits in the code, print the sequence of gray code. A gray code sequence must begin with 0.

  1. Convert the binary number into gray code: b(n) = n^(n>>1)
  2. Other solution.

91. Decode Ways

A message containing letters from A-Z is being encoded to numbers using the following mapping: A->1 … Given a non-empty string containing only digits, determine the total number of ways to decode it.

  1. Well, I met this one on the apple on campus interview…. we can easily come up with a recursive method.

  2. However, DP is a better solution. Use a vector to store that how many decode ways each range from the start to end index has, so that we can save lots of computation.

  3. The dp function is v[i] = v[i-1] + v[i-2], v[0] is assigned to 1, v[i-1] represents that i is for a single character, and v[i-2] represents that i and i-1 combines for a character, note that there is v[i-1] + v[i-2].

93. Restore IP Addresses

Given a string containing only digits, restore it by returning all possible valid IP address combinations.

In fact, we only have at most $ C^3_11 $ cases, so it will be fine to do Bruce Forte. So we need to enumerate all of the cases in four while loops, and use stoi to transfer the string into int.

128. Longest Consecutive Sequence

Given an unsorted array of integers, find the length of the longest consecutive elements sequence. Your algorithm should run in O(n) complexity.

Use a hash map to store the length of the sequence, in the left and right end index of the sequence, smart!

134. Gas Station

There are N gas stations along a circular route, where the amount of gas at station i is gas[i]. You have a car with an unlimited gas tank and it costs cost[i] of gas to travel from station i to its next station (i+1). You begin the journey with an empty tank at one of the gas stations. Return the starting gas station’s index if you can travel around the circuit once in the clockwise direction, otherwise return -1.

  1. Compute the difference between cost and gas
  2. Use a loop to compute the sum of the differences, if sum>=0 there will be a solution, otherwise not
  3. In the same loop, compute a temporary sum, when the sum is negative, clear the sum and restart from the next index.
  4. Why? If the temporary sum is negative, it means there is no solution from the original starting index to this index.
  5. A weird solution, requiring explanation.

151. Reverse Words in a String

Given an input string, reverse the string word by word.

  1. Reverse the string, and do split, push back reversed words into res.
  2. Don not insert char in the front of string, which makes it really slow.
  3. Transfer the size_t into int first, because unsigned int may get out of the range.

161. One Edit Distance

Given two strings s and t, determine if they are both one edit distance apart.

  1. If sizes of the two strings are equal, they should only differ in one character.
  2. If the sizes differ more than 1, return false. Otherwise, we can always treat string s as the smaller string which is only one character less then string t, so that we can only consider about delete. In this way, the substrings after the different character in string s and t should be the same.

165. Compare Version Numbers

Compare two version numbers version1 and version2.
If version1 > version2 return 1; if version1 < version2 return -1;otherwise return 0.

  1. Detect the sub version of both v1, v2 in a loop and compare them.
  2. If one of the string end, set the version to 0, keep comparing it with the other

179. Largest Number

Given a list of non negative integers, arrange them such that they form the largest number.

  1. Use custom comparator.
  2. To compare two string s1, s2, return s1+s2>s2+s1, smart!
  3. Be careful of 0,0 -> 0, X->00.
