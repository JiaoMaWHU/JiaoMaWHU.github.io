Leetcode Summary - Others

Posted on 2019-10-05 | In Algorithms |

This post records my solutions of leetcode in the Math, Stack, Queue, Bit-manipulation, Union-Find, Design tag.

29. Divide Two Integers

Given two integers dividend and divisor, divide two integers without using multiplication, division and mod operator. Return the quotient after dividing dividend by divisor. The integer division should truncate toward zero.

  1. Use regular minus will be too slow;
  2. Use binary search, square the division everytime.

50. Pow(x, n)

Implement pow(x, n), which calculates x raised to the power n (x^n).

Do 1, x , x^2, x^4…., making x grow exponentially to meet the time requirements.

137. Single Number II

Given a non-empty array of integers, every element appears three times except for one, which appears exactly once. Find that single one.

  1. For all integers in the vector, sum up 1 for every bit from the no.0 to the no.31, then mod the sum to 3, and add up all the bit together, therefore we get the final value.
  2. Other solutions in LC seems unclear.

146. LRU Cache

Design and implement a data structure for Least Recently Used (LRU) cache. It should support the following operations: get and put.

  1. Use a list to store , a map to store .
  2. Use splice to change a existed element to the front of the list in put.
  3. For put function, if the key exists, splice, otherwise pop_back and erase the map key and then create a new element in list and map.

155. Min Stack

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

  1. Use a stack to store data, the other to store minimum elements
  2. Be careful, when a element is smaller than the top of min stack or the min stack is empty, push it.

168. Excel Sheet Column Title

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

  1. Be careful, the scale is from 1 to 26, it does not have a zero representation, so we can not easily follow the decimal operation
  2. if n mod 26 == 0, add Z to the result, otherwise add the corresponding character
  3. n -= (26 or n%26), n /= 26

200. Number of Islands

Given a 2d grid map of ‘1’s (land) and ‘0’s (water), count the number of islands. An island is surrounded by water and is formed by connecting adjacent lands horizontally or vertically. You may assume all four edges of the grid are all surrounded by water.

  1. We can use DFS or BFS, but union-find is a typical solution. Be care that we need the rank vector to decide which tree as the parent in the union process.
  2. For the island number, we can initial a number count as the number of ‘1’ in input. Then, when two elements have different parents in union, minus the count by 1. At the count is the number of islands.

225. Implement Stack using Queues

  1. For push: push in the new element, pop and push everything again except the new element
  2. So that, we can easily implement pop, top function.

232. Implement Queue using Stacks

  1. Use two stack, the first to store
  2. When pop or top is called, and the second stack is empty, pop every elements in the first stack into the second.
