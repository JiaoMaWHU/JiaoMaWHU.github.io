137. Single Number II

Given a non-empty array of integers, every element appears three times except for one, which appears exactly once. Find that single one.

  1. For all integers in the vector, sum up 1 for every bit from the no.0 to the no.31, then mod the sum to 3, and add up all the bit together, therefore we get the final value.
  2. Other solutions in LC seems unclear.

146. LRU Cache

Design and implement a data structure for Least Recently Used (LRU) cache. It should support the following operations: get and put.

  1. Use a list to store , a map to store .
  2. Use splice to change a existed element to the front of the list in put.
  3. For put function, if the key exists, splice, otherwise pop_back and erase the map key and then create a new element in list and map.

225. Implement Stack using Queues

  1. For push: push in the new element, pop and push everything again except the new element
  2. So that, we can easily implement pop, top function.

232. Implement Queue using Stacks

  1. Use two stack, the first to store
  2. When pop or top is called, and the second stack is empty, pop every elements in the first stack into the second.
