70. Climbing Stairs

You are climbing a stair case. It takes n steps to reach to the top. Each time you can either climb 1 or 2 steps. In how many distinct ways can you climb to the top? Note: Given n will be a positive integer.

The dp equation is dp[n] = dp[n-1] + dp[n-2]

72. Edit Distance

Given two words word1 and word2, find the minimum number of operations required to convert word1 to word2. You have the following 3 operations permitted on a word: Insert a character, Delete a character, Replace a character

  1. Build a 2D dp array dp[size1][size2], where dp[i][j] means the edit distance of string1[0…i] with string2[0…j]
  2. Initialize dp[0][j] = j, from j=0 -> j=size2, so does dp[i][0]
  3. if string1[i] == string2[j], dp[i][j] = dp[i-1][j-1], else dp[i][j] = 1 + min(dp[i-1][j] (delete string1[i]), dp[i][j-1] (insert string2[j] to string1), dp[i][j] (replace)).

131. Palindrome Partitioning

Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s.

  1. Use the same 2D dp array idea as 5. Longest Palindromic Substring.
  2. Then use DFS to get all possible results.
  3. TC: O(): ? \(C_{n-1}^1 + C_{n-1}^2 + … + C_{n-1}^{n-1}\), SC: O(n^2)

132. Palindrome Partitioning II

Given a string s, partition s such that every substring of the partition is a palindrome. Return the minimum cuts needed for a palindrome partitioning of s.

  1. Build the same 2D dp array as 5. Longest Palindromic Substring, also build a dp array S to records the minimum cuts needed in substring 0-i
  2. When determine dp[j][i], if dp[j][i] is true, also determine S[i] = min(S[i], dp[0][j-1]+1) (j>0, else j==0, S[i] = 0).

139. Word Break

Given a non-empty string s and a dictionary wordDict containing a list of non-empty words, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

Note: The same word in the dictionary may be reused multiple times in the segmentation. You may assume the dictionary does not contain duplicate words.

  1. Use the same DP idea as in 132. Palindrome Partitioning II.
  2. Build a 1D DP array, DP[i] means if substring 0-i can be segmented into words in dictionary.

123.Best Time to Buy and Sell Stock III

Say you have an array for which the ith element is the price of a given stock on day i. Design an algorithm to find the maximum profit. You may complete at most two transactions. Note: You may not engage in multiple transactions at the same time (i.e., you must sell the stock before you buy again).

  1. Use DP : local[i][j], global[i][j] to represent the profit. local[i][j] means the NO. j purchase is sold at time i. global[i][j] is the max profit at time i, where the last stock may be smaller than j.
  2. The update for local is local[i][j] = max(local[i-1][j], g[i-1][j-1]) + (prices[i] - prices[i-1]). Why? g[i-1][j-1] means that the last stock is purchased at time i-1, and I want to sell it right, so it is the diff profit plus the profit before. local[i-1][j] means that the last stock was purchased before i-1 and selled at time i-1, but I want to sell it right now.
  3. global[i][j] = max(global[i-1][j], local[i][j])

198. House Robber

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed, the only constraint stopping you from robbing each of them is that adjacent houses have security system connected and it will automatically contact the police if two adjacent houses were broken into on the same night. Given a list of non-negative integers representing the amount of money of each house, determine the maximum amount of money you can rob tonight without alerting the police.

  1. Use a DP array dp[i], represent the max money when the robber is at house i.
  2. dp[i] = max(dp[i-1], dp[i-2]+money[i])

221. Maximal Square

Given a 2D binary matrix filled with 0’s and 1’s, find the largest square containing only 1’s and return its area.

  1. Use 2D dp to denote the maximum square side length, if this point is the right bottom of the square. So we have the dp equation: dp[i][j] = min(dp[i][j-1], dp[i-1][j], dp[i-1][j-1]) + 1, (if this point is ‘1’). The insight is, the other bigger two points supports the smallest point to be the right bottom of a square.
  2. We can even use a 1D dp array by using a variable PREV, which represents the value of dp[i-1][j-1]. Then, the equation is dp[j] = min(dp[j], dp[j-1], prev) + 1, and we also need to update the PREV at the end.
