152. Maximum Product Subarray

Given an integer array nums, find the contiguous subarray within an array (containing at least one number) which has the largest product.

dp

  • max(k) = max( max(k-1) A[k], A[k], min(k-1) A[k] )
  • min(k) = min( min(k-1) A[k], A[k], max(k-1) A[k] )

STL & Data Structure

128. Longest Consecutive Sequence

Given an unsorted array of integers, find the length of the longest consecutive elements sequence. Your algorithm should run in O(n) complexity.

  • Can’t use a set, whose insert function is O(logn) complexity.
  • Unordered_set is ok. But you need to erase element in case of redundant compare.
  • Unordered_map is also ok. But the [ ] operator of Unordered_map will creates a new element, so do not mix it up with the count function.
  • Bitset is ok, which is another version of unordered_set using less memory.
